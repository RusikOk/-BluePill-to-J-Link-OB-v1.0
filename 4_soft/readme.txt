последовательность создания новой версии прошивки программатора для МК CS32F103

1) зашить прошивку 4_JLOB103V1_bootloader.bin
2) подключаем J-Link к компьютеру по USB и запускаем J-Link Commander, а он уже установит актуальную версию прошивки
3) запустить утилиту JLink.exe из старой версии драйверов c:\Program Files (x86)\SEGGER\JLink_V500\
4) установить новый серийный номер коммандой "exec SetSN=20210420"

опять что-то поменялось(

1) зашиваем прошивку загрузчика 4_JLOB103V1_bootloader.hex в контроллер со 128кБ памяти
2) подключаем по USB
3) запускаем самую свежую версию JLink.exe, и обновляем прошивку
4) несколько раз отключаем программатор по USB и обратно включаем
5) подключаемся по SWD к контроллеру с полной прошивкой
6) вычитываем 64кБ (Size 0x10000) и сохраняем в файл
7) записываем с файла в CS32F103
8) копируем 8 байт с адреса 0x08004000 в буффер обмена и вставляем на адрес 0x08000000
9) устанавливать новый серийник ЗАПРЕЩЕНО!!! пусть остается -1
10) готовая прошивка от 2021-10-27 с названием 5_2021-10-27.hex
